# Giftos-Website

Giftos is a modern, responsive website built with HTML, CSS, and Bootstrap, designed to be your one-stop destination for finding the perfect gifts for any occasion.

Responsive Design: Giftos is optimized for desktop, tablet, and mobile devices, ensuring a seamless browsing experience across all screen sizes.

Bootstrap Integration: Leveraging the power of Bootstrap, Giftos boasts clean, elegant layouts and responsive components for a polished look and feel.
